class와 id 차이

  • 클래스와 아이디 쓰이는 곳

    • 클래스 : 여러가지 스타일링을 한꺼번에 적용하고 싶을 때


    아이디 : 스타일링을 한가지만 적용하고 싶을 때

  • 이름을 불러오는 방식

    • 클래스 : 클래스를 불러올 때 클래스명 앞에 마침표(.)를 찍어준다.


    아이디 : 아이디를 불러올때는 아이디값 앞에 샵(#)을 표시해준다.

  • 중복 사용 여부

    • 클래스 : 클래스는 우복 사용이 가능하여, 동일한 클래스명을 페이지 여러곳에 사용가능하다.


    아이디 : 아이디는 중복으로 사용이 안되어서, 페이지에서 딱 한번만 사용해야 한다.

    플렉스박스와 그리드의 차이점

  • 플렉스 박스

    • 1차원으로 수평, 수직 영역 중 하나의 방향으로만 레이아웃을 나눌 수 있음

  • 그리드

    • 2차원으로 수평 수직을 동시에 레이아웃을 나눌 수 있음
